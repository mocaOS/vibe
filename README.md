# The Vibe Studio

**MOCA's City Center for Crypto Art, AI Agents, and Creative Innovation**

The Vibe Studio is the nexus for everything the Museum of Crypto Art has created, is creating, and will create in the future. It's MOCA's cumulative UI/UX overhaul‚Äîa navigable network of applications, tools, and experiences that bring together the Museum's collections, the Art DeCC0s agentic ecosystem, and community-built innovations into one unified home.

This open-source codebase serves as both MOCA's digital headquarters and a framework for other web3 communities to build their own application ecosystems. By integrating seamlessly with [mocaOS](https://github.com/mocaOS/museum) and leveraging CC0 data around Art DeCC0s, the Vibe Studio enables developers to create compelling experiences that fuel storytelling and creativity in highly accessible ways.

## The Technology Behind It

### The Art DeCC0s Codex

At the heart of Vibe Studio's creative potential lies the Art DeCC0s Codex‚Äîa near-limitless environment of personality data created through the technical brilliance of Da√ØmAlYad. Using multimodal generation and peerless problem-solving, each of the 10,000 Art DeCC0s has been imbued with unique, interactive characteristics.

The Codex contains:
- Each DeCC0's name, biography, and favorite artist
- Their writing style, hometown, and self-conceptualization
- Appearances, predilections, and sensitivities
- Inherent knowledge bases and sample conversations
- Technical information for agentic launchers like ElizaOS

This massive dataset serves as both a way for DeCC0 holders to understand their NFTs' personalities and a framework for ambitious developers to create better, more complex, more alive agents.

### Integration with mocaOS

The Vibe Studio plugs directly into [mocaOS](https://github.com/mocaOS/museum), allowing developers to leverage:
- CC0 datapoints around Art DeCC0s (media, codex data, etc.)
- The MOCA Library‚Äîan ever-growing knowledge graph of web3 arts and culture
- AI-powered retrieval systems for agentic knowledge
- ElizaOS character files for agent creation

## Running Vibe Studio Locally

To run Vibe Studio on your local machine:

1. Ensure you have Node.js installed (version 12 or higher)
2. Clone this repository
3. Navigate to the project directory
4. Install dependencies:
   ```bash
   npm install
   ```
5. Start the server:
   ```bash
   npm start
   ```
6. Open your browser and navigate to `http://localhost:8080`

To stop the server, press `Ctrl+C` in the terminal where it's running.
